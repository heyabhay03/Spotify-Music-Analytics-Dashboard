# 🎵 Spotify Music Analytics Dashboard

## 📌 Project Overview

The **Spotify Music Analytics Dashboard** is an interactive Business Intelligence solution developed using **Power BI** to analyze Spotify's Top 50 music data. The dashboard provides valuable insights into song popularity, artist performance, album trends, and listening patterns through dynamic visualizations and KPI-driven analytics.

This project demonstrates expertise in **Data Analysis, Data Modeling, DAX, Power Query, and Dashboard Design**, enabling users to explore music industry trends and make data-driven decisions.

---

## 🚀 Key Features

### 📊 Interactive Dashboard Navigation

* Multi-page dashboard experience
* Home, Overview, Artist, and Songs pages
* Dynamic page navigation using bookmarks and buttons

### 🎯 KPI Tracking

* Total Songs
* Total Artists
* Average Popularity Score
* Average Song Duration

### 🎵 Song Analytics

* Top-performing songs
* Popularity distribution analysis
* Song trend monitoring
* Duration analysis

### 🎤 Artist Analytics

* Most popular artists
* Artist-wise song contribution
* Popularity comparison across artists

### 💿 Album Analytics

* Album type distribution
* Single vs Album comparison
* Popularity analysis by album category

### 📈 Trend Analysis

* Quarterly popularity trends
* Year-wise song releases
* Seasonal music performance insights
